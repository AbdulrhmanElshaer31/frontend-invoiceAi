| Usage                        | Color     | Reason                                          |

| **Secondary Text**           | `#1D546D` | Darker than teal, suitable for headings/details |
| **Buttons / Call-to-Action** | `#5F9598` | Eye-catching, conveys activity and attention    |
| **Button Hover / Highlight** | `#1D546D` | Darker than main button color for hover effect  |
